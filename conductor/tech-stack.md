# Tech Stack

## Language
- **TypeScript:** Primary programming language for both frontend and potential server-side logic (e.g., Edge Functions).

## Frontend Framework
- **Angular:** Modern framework with Standalone components and Signals for reactive state management.
- **Ant Design (ng-zorro-antd):** UI component library for building professional interfaces.
- **Tailwind CSS v4:** Utility-first CSS framework for modern styling.
- **Lucide Icons:** Consistent iconography library.

## Backend & Database
- **Supabase (PostgreSQL):** Robust relational database management system, migrating from Firebase Realtime Database.
- **Supabase Auth:** Integrated authentication solution for secure user access.
